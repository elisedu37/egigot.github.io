2017 - 2019

Notre dame la riche - Tours (37)

Durant ces deux années, j'ai passé mon baccalauréat scientifique. A ce moment-là, j’ignorais encore quelle orientation je prendrai exactement. Je connaissais seulement le domaine qui m’intéressait : l'informatique au sens large. C'est la raison pour laquelle, j’ai choisi l’option informatique au lycée. J'ai alors eu un déclic. Je souhaitais travailler dans le web, que ce soit dans le design ou dans le développement.
